# Chrome Booking System Plugin Extension

## Overview

Chrome Booking System Plugin Extension automates selected slot-booking and container-monitoring workflows for supported port systems. It includes queue management, retry handling, optional portal auto-login, optional browser and email notifications, and account-based feature access.

## Main Capabilities

- Automates booking workflows for supported port websites.
- Maintains booking queues and retries failed operations.
- Supports DCT, BCT, and GCT-related workflows available to the signed-in account.
- Offers optional portal auto-login using credentials stored locally in the browser.
- Offers optional browser notifications and optional email notifications.
- Displays current queue and processing status in the extension popup.

## Installation

### Chrome Web Store

Install the published build from the Chrome Web Store listing for this extension.

## Data and Privacy Summary

- The extension stores operational data locally in the browser, including settings, queue state, notification preferences, and optional portal auto-login credentials.
- The extension uses third-party backend infrastructure for account authentication, feature-access control, analytics, and diagnostics.
- The extension can send optional notification emails through a third-party email delivery provider when email notifications are enabled by the user.
- The extension interacts only with supported port websites required for booking and container-monitoring workflows.
- Full details are provided in [PRIVACY_POLICY.md](PRIVACY_POLICY.md).
